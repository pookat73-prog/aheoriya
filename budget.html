☁ 클라우드 동기화
여러 기기에서 가계부를 공유하려면 Firebase 무료 플랜을 1회 설정해야 해요. 설정 방법 보기
1. Firebase Console에서 프로젝트 생성
2. Build → Authentication → Sign-in method → Google 활성화
3. Build → Firestore Database → 프로덕션 모드 → 위치 선택 → 만들기
4. Firestore의 Rules 탭에 아래 규칙 붙여넣기: 
rules_version='2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /budgets/{uid} {
      allow read,write: if request.auth.uid == uid;
    }
  }
}
5. 프로젝트 설정(⚙) → 일반 → "내 앱"에서 웹 앱(</>) 추가 → 닉네임 입력
6. 표시되는 firebaseConfig = {...} 객체를 통째로 복사
7. 인증 → Settings → 승인된 도메인에 pookat73-prog.github.io 추가
8. 아래 칸에 붙여넣기 → 저장
로그인하면 모든 기기에서 같은 데이터가 자동 동기화됩니다.
